1 · Algorithm & options

Queue 1 has the highest priority: an arrived process in a higher queue preempts lower queues. FCFS / SJF run non-preemptively inside their queue; SRTF and Priority re-evaluate every time unit; RR uses its own quantum. Assign each process a queue number in the table below.

Formula Reference

Per-process formulas

CTCompletion Time — when the process finishes its full burstread from Gantt chart
TATTurnaround TimeTAT = CT − AT
WTWaiting TimeWT = TAT − (BT + I/O Time)
RTResponse Time (first CPU allocation)RT = First Start − AT

Overall formulas

ThroughputCompleted work per timeNumber of processes ÷ Last CT
OverheadContext switching rateNumber of switches ÷ Last CT
EfficiencyUseful CPU percentage(1 − Overhead) × 100
AveragesAvg TAT / WT / RTSum ÷ Number of processes

A context switch is counted whenever the CPU changes directly from one process to a different process. Idle → process is not counted as a process-to-process switch.
